## About

This repository contains my solutions, agents, and experiments built throughout the **AI_devs 4** course - a 5-week intensive program focused on building production-ready AI systems.

The course covers the full stack of modern AI engineering: from prompting and tool use, through RAG pipelines and knowledge bases, to autonomous agents and deployment.

---

## Course Structure

| Week | Focus |
|------|-------|
| 1 | Language model programming, function calling, MCP servers, multimodal |
| 2 | Context management, RAG systems, document processing, agent design |
| 3 | Non-determinism handling, evaluation, feedback loops |
| 4 | Active AI collaboration, knowledge bases, enterprise patterns |
| 5 | Architecture, production tooling, deployment |

---

## Goals

- Build real, working AI agents - not just demos
- Understand context and prompt engineering
- Implement RAG pipelines from scratch
- Learn how to evaluate and debug LLM-based systems
- Ship production-grade solutions
